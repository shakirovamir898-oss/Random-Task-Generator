# Random Task Generator

**Автор:** Шакиров Амир Рамилевич

## Описание
Приложение "Random Task Generator" позволяет генерировать случайные задачи из предопределенного списка или добавленных пользователем. Поддерживает фильтрацию по типу задачи (учёба, спорт, работа) и сохранение истории генерации.

## Функционал
- Генерация случайной задачи из списка
- Добавление пользовательских задач с проверкой на пустоту
- Фильтрация истории по типу задачи
- Сохранение истории и списка задач в JSON-файлы
- Интуитивно понятный графический интерфейс

## Примеры использования

### 1. Генерация задачи
1. Запустите приложение
2. Нажмите кнопку "Сгенерировать случайную задачу"
3. В истории появится новая запись с задачей и её типом
4. Приложение покажет всплывающее уведомление с деталями задачи

### 2. Добавление задачи
1. Введите текст задачи в поле "Текст задачи"
2. Выберите тип задачи из выпадающего списка
3. Нажмите "Добавить задачу"
4. Если поле пустое, появится предупреждение об ошибке

### 3. Фильтрация истории
1. Выберите тип задачи в фильтре ("учёба", "спорт" или "работа")
2. История автоматически отфильтруется
3. Для сброса фильтра нажмите кнопку "Сбросить"

### 4. Сохранение данных
- Все данные автоматически сохраняются при закрытии приложения
- При следующем запуске сохраненные задачи и история будут загружены

## Тестирование
1. **Проверка ввода:**
   - Попробуйте добавить пустую задачу → должно появиться предупреждение
   - Убедитесь, что ввод с пробелами тоже отклоняется

2. **Генерация задач:**
   - Сгенерируйте 5-7 задач разных типов
   - Проверьте корректность отображения в истории

3. **Фильтрация:**
   - Сгенерируйте задачи всех типов
   - Примените фильтр по каждому типу
   - Убедитесь, что отображаются только задачи выбранного типа

4. **Перезапуск приложения:**
   - Добавьте задачи и сгенерируйте несколько задач
   - Закройте приложение
   - Перезапустите и проверьте сохранение данных
